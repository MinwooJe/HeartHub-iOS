**HeartHub - 커플만의 공간, 커플들끼리의 커뮤니티**
- 커플 공간에서 연인과의 추억들을 기록할 수 있어요
- 커뮤니티에서 다른 커플들의 일상, 룩, 데이트 코스들을 공유할 수 있어요
